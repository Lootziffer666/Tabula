# Tabula – Map. Move. Link.

Tabula ist jetzt auf **Storage Intelligence + Safe Relocation** fokussiert:

- **Map:** schwere oder chaotische Speicherorte sichtbar machen
- **Relocate:** große, aber nicht sofort löschbare Ordner sicher umplanen
- **Links:** verwaltete Junction-/Symlink-Ziele im Blick behalten
- **History:** Relocation-Ledger lokal exportieren

## Start

```bash
pip install -r requirements.txt
python tabula.py
```

## MVP-Scope

Tabula ist **kein** Debloat-/Privacy-/Allzweck-Maintenance-Host mehr. Der v1-Fokus ist:

1. Speicherorte erkennen
2. Risiko & Empfehlung zeigen
3. sichere Relocation-Vorschau liefern
4. geplante Links nachvollziehbar dokumentieren
